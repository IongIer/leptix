## How to run

1. Clone this repository

```
git clone https://github.com/Upbolt/leptos_primitives.git
cd leptos_primitives/examples/ssr-with-actix-tailwind
```

2. Run [cargo-leptos](https://github.com/leptos-rs/cargo-leptos)

```
cargo leptos serve
```
